Apache Flink 文档

Apache Flink 是一个在无界和有界数据流上进行状态计算的框架和分布式处理引擎。 Flink 已经可以在所有常见的集群环境中运行，并以 in-memory 的速度和任意的规模进行计算。

试用 Flink

如果您有兴趣使用 Flink, 可以试试我们的教程:

  • Local Installation
  • DataStream API 进行欺诈检测
  • Table API 构建实时报表
  • PyFlink 教程
  • Flink 游乐场

学习 Flink

  • 操作培训 包含了一系列的课程和练习，提供了对 Flink 的逐一介绍。

  • 概念透析 介绍了在浏览参考文档之前你需要了解的 Flink 知识。

获取 Flink 帮助

如果你遇到困难了, 可以在 社区寻求帮助。值得一提的是，Apache Flink 的用户邮件列表一直是 Apache 项目里面最活跃的之一，也是一个快速获得帮助的好途径。

探索 Flink
